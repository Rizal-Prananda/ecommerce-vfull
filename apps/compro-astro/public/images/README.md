# Product Images Directory

Simpan gambar-gambar berikut ke folder ini:

## Required Images

1. **hero-model.jpg** - Gambar model fashion untuk hero section (1200x900 px atau lebih)
2. **avatar-1.jpg** - Avatar untuk section rating (96x96 px)
3. **avatar-2.jpg** - Avatar untuk section rating (96x96 px)
4. **avatar-3.jpg** - Avatar untuk section rating (96x96 px)
5. **collection-new-arrivals.jpg** - Gambar koleksi New Arrivals (600x400 px)
6. **collection-best-seller.jpg** - Gambar koleksi Best Seller (600x400 px)
7. **collection-mens.jpg** - Gambar koleksi Men's Edit (600x400 px)
8. **collection-womens.jpg** - Gambar koleksi Women's Edit (600x400 px)

## Notes

- Gambar produk masih menggunakan sumber eksternal (CMS atau picsum.photos) dan dapat dikonfigurasi di halaman detail
- Semua referensi URL eksternal dalam `index.astro` telah diubah untuk menunjuk ke jalur lokal `/images/`
